# NAG Difference Core Theory

This document introduces two complementary tracks:

1. **Endpoint-cancellation track**: pairwise differences cancel shared terms.
2. **Chain-certificate track**: finite path/antichain certificates for admissible transitions.

Important caution:
- The hopfion case study is a motivating benchmark and does **not** itself prove NAG Difference.
- This repository does **not** claim a full formal proof of Erdős #1196.
