# Case Study: Erdős #1196 (Inspiration Only)

This repository uses Erdős problem #1196 as a structural inspiration for finite chain certificates.

- Integers are modeled as states.
- Divisibility-oriented transitions define a finite sub-Markov chain.
- Primitive sets are treated as antichains in the divisibility order.

The chain-certificate utilities check antichain hit-at-most-once behavior on finite paths and compute finite-horizon hitting probabilities.

> This repository does **not** provide a proof of Erdős #1196 and does not claim formalization of the full theorem.
