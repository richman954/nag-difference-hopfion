# Erdős #1196 Inspired Finite Toy Model

This note uses Erdős #1196 only as inspiration for a finite toy model.

- Integers are treated as states.
- Divisibility defines a transition order.
- Primitive sets are antichains in that order.
- A sub-Markov chain assigns visiting probabilities.
- Admissible paths hit a primitive set at most once.

This motivates a NAG Difference **chain-certificate** idea for finite-state reasoning.

Caution:
- We do **not** claim this repository formalizes Erdős #1196.
- We do **not** claim the hopfion experiment proves NAG Difference.
